# AI Shopping Assistant Agent

An AI-powered shopping assistant that understands user queries and recommends the best products using LLM agents.

## Features
- Natural language product search
- AI-based recommendation engine
- Review sentiment analysis
- MERN + Python AI architecture

## Tech Stack
- React (Frontend)
- Node.js (Backend)
- Python (AI Agent)
- MongoDB

## Setup

### 1. Clone repo
git clone https://github.com/your-username/ai-shopping-agent

### 2. Install dependencies
cd client && npm install
cd server && npm install
cd agent && pip install -r requirements.txt

### 3. Run project
npm run dev
